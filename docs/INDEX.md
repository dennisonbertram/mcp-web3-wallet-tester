# MCP Web3 Wallet Tester - Documentation

> Last updated: 2025-12-21

## Overview

This directory contains all documentation for the MCP Web3 Wallet Tester. The wallet tester enables automated testing of Web3 dApps by acting as a programmable Ethereum wallet that LLMs can control via MCP tools.

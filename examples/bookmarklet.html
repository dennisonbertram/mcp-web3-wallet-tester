Installation

  1. Drag the button below to your bookmarks bar
  2. Start the wallet server: npm start
  3. Visit a dApp that supports EIP-6963 wallet selection
  4. Click the bookmarklet, then try selecting the wallet
üì• Web3 Test Wallet

How It Works

  1. The bookmarklet loads provider.js from the wallet server
  2. The script creates an EIP-1193 provider connected via WebSocket
  3. It announces itself via EIP-6963
  4. Also sets window.ethereum if possible

For Playwright Testing (Recommended)

For proper automated testing, inject the provider before page load:

// In your Playwright test
const providerScript = await fetch('http://localhost:3001/provider.js').then(r => r.text());
await page.addInitScript(providerScript);
await page.goto('https://app.uniswap.org');

Direct Access

After injection, you can access the wallet directly:

// In browser console
window.__web3TestWallet__.request({ method: 'eth_requestAccounts' })
window.__web3TestWallet__.request({ method: 'eth_chainId' })
